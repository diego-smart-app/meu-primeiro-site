Como usar o GitHub

O GITHUB é uma ferramenta poderosa para compartilhamento de projetos e também de armazenamento/gerenciamento de versões de softwares criados. Pode ser pago ou gratuito, dependendo da aplicação, cada repositório pode ser privado ou público, existem algumas limitações para a opção grátis, porém com ela é possível fazer muitas coisas. A seguir vamos ver o passo a passo para usar esta ferramenta.

  1. Baixar o GIT

    Para usar o GITHUB temos que ter em mente que precisamos do GIT, ela é uma ferramenta auxiliar ao GITHUB. Para isso basta baixar o programa aqui e efetuar o download de acordo com seu sistema operacional.

  2. Instalando o GIT

    Para instalar basta seguir o padrão para a maioria dos softwares, é só clicar em NEXT, NEXT, etc. E o software estará pronto para o uso.

  3. Criando uma conta no GitHub

    Como o GitHub é um repositírio online para guardar nossos projetos, precisamos fazer um cadastro no site, se você não tem, é só clicar aqui, fazer seu cadastro e abrir sua conta gratuita.

  4. Criando um repositório

    Como falado anteriormente, o repositório é uma “pasta online” onde nosso projeto será armazenado. Para criar um novo repositório, faça login na sua conta do GitHub, clique em CREATE A NEW REPOSITORY (caso não funcione clique em NEW):



    Depois é só dar um nome e uma descrição. O nome deve ser específico, por exemplo um projeto de aplicativo mobile gerenciador de finanças pessoais, o nome pode ser: aplicativo-gerenciador-financeiro. Procure também colocar uma descrição (arquivo README.me).

  5. Clonar pasta no PC

    Agora que temos o repositório online, podemos fazer uma cópia da mesma pasta no nosso computador, pra fazer isso clique em CODE e copie o link:



    A imagem acima mostra a tela exibida pra fazer uma cópia do link. Depois de copiar temos que executar usando o passo a seguir.

  6. Usando o GIT

    Agora vamos usar o programa GIT, primeiro abrimos o CMD do computador (usando o Windows) e colocamos o comando
    git clone link_que_voce_copiou



    Agora temos uma cópia do nosso repositório em nosso computador.

  7. Adicionar os arquivos no repositório online

      8. Antes de mais nada temos que criar todo o projeto dentro da nossa pasta, depois efetuar os camandos:

      • cd nome-repositorio
      • git add
      • git commit -m
      • git push

🎉 Parabéns seu projeto está no GitHub 🎉

Atualizando o repositório

Vamos ver os comandos e entender cada um.

  1. Abrir o cmd
  2. Entrar na pasta que está a ser sincronizada como repositório
    • cd pasta-repositório
  3. Verificar status de modificação do arquivo:
    • git status
  4. Reconhecer pelo gerenciador local (GIT):
    • git add .
  5. Adicionar no gerenciador Git:
    • git commit - "descrição ou versão"
  6. Verificar se é a última modificação de fato (verifica se é a última alteração que você fez, pois pode ter o caso de outros também estar atualizando o projeto):
    • git pull origin main
  7. Atualizar repositório remoto do GitHub (esse passo exijirá o login e senha):
    • git push origin main

Detalhe importante

Quando se trata de um projeto com vários programadores e você está a modificar algo, o primeiro comando antes de tudo é usar o:

  • git pull origin main
Nesse caso a mensagem deverá ser: Already up to date. Caso contrário deverá ser atualizado para então ser modificado, pois poderá inutilizar o trabalho de quem estava a mexer.
